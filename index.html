Sobre SCRRIS®

SCRRIS es un software de control de membresia, ingresos, gastos, cursos, agenda,notificaciónes de cumpleaños, etc. desarrollado especialmente para iglesias.

Facil instalación

La instalación del software se realiza de forma remota, solo se requiere de conexión a internet para poder realizar la implementación del sistema.

Bajos requisitos

SCRRIS no requiere de requisitos altos para poder ser utilizado, solo requiere de una PC, notebook, netbook o laptop que posea como SO Windows y conexión a internet estable.

Múltiples funciones

El sistema posee varias funciones, de las cuales se destaca en su nueva actualización, la posibilidad de montar en tan solo unos minutos una radio Online por ZenoMedia sin necesidad de generar licencias ni equipamientos para transmisión 24/7 las 24hs. Puedes escuchar tu radió online desde tu smartphone simplemente escaneando un código QR que el sistema genera.

Módulos

SCRRIS Software tiene variós módulos en los cuales puedes encontrar apartados para los miembros, proveedores, ingresos,gastos, Radio Online, usuarios, agenda, cursos, correos, etc.

SCRRIS® Software

Nuestro propósito es que la iglesia pueda llevar una buena administración y control de sus miembros, ingresos, gastos, cursos, etc. Sin necesidad de mucho esfuerzo ni tener mucho conocimiento de técnologia. Acercamos a cada iglesia (ya sea de muchos miembros o pocos), la posibilidad de un orden cómodo, sencillo y eficaz.

SCRRIS es un software sumamente sencillo y eficaz desarrollado para iglesias.
